Columns Block Demo

This page demonstrates the 2-column layout that renders side-by-side on desktop.

Left Column

This is the left column content. On desktop (900px+), this appears side-by-side with the right column. On mobile, it stacks vertically.

You can add multiple paragraphs, headings, lists, and other content here.

Right Column

This is the right column content. Both columns have equal flex: 1 width at 900px+, with a 32px gap between them.

Images in columns are handled specially with the columns-img-col class.
