# Edge Agentify

Convert Edge Delivery blocks into AI-readable structures.

## Steps
1. Identify block purpose
2. Extract:
   - entity
   - intent
   - key fields
3. Inject JSON into DOM using <script type="application/json">

## Rules
- Do not duplicate DOM
- Keep JSON minimal
- Map directly from content

## Output
- Updated block code
- AI JSON structure
